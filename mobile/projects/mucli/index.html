Project

muCLI

Python, yt-dlp, yt-music-api

muCLI is a command line music playing app that I built during hacknight, an event hosted by my college's open source club. muCLI allows users to listen to music of their choice while showing the acoustic wave patterns, and the lyrics of the song. The project used numpy to convert the song data into height bars. It utilized yt-dlp for downloading the songs, and yt music API for finding the correct song based upon user input. To ensure smooth and quick experience, the song is actually downloaded in chunks in this application, this makes use of multi threading features of python. I posted this project of mine on reddit, where it gained 1k+ upvotes and 65k+ views.
