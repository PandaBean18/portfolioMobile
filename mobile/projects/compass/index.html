Project

Compass

Python, Flask, PHP

Compass is a web app that helps users find the best route between any two points. The app allowed users to select from "fastest", "cheapest" and "eco friendly" options and were given a route with estimated time and cost. The app incorporated data from Delhi bus routes. It also used data from Delhi metro. When this app was built, an official API for Delhi Metro Rail Corporation did not exist, so we relied on web scraping to try and fid the time estimates and metro fares.

This project was made for a hackathon in which we won second place.
