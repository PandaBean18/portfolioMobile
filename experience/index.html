I interned at Auto Verdure last year as a software development intern for a period of two months from May till July. During this internship I worked heavily on the company website's backend architecture, helped in setting up the company's database on firebase, facilitating CRUD functionality by writing an API for the website, and also worked on improving the website UI a few times. I also worked a lot with platforms such as Google Cloud, which I used to set up Google's One Tap sign in, Cloudinary, which I used to host images of the website to try and make use of their AI features, Resend, which was used to automate email sending, and Razorpay, who's payment gateway was employeed into the website. I also worked with frontend libraries such as react and three.js. Since the website was built using Next.js, I worked very closely with that as well, and understood a lot about writing APIs in the framework. I was also part of an effort to improve website performance by employing client side and server side caching, which helped me gain valuable experience in working with systems like Redis and Redux. I was also responsible for implementing session token based authentication and authorization in the website.
