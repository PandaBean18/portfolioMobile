Project

beanVC

C++

beanVC is version control management system that I built to improve upon my C++ skills, it was also my end semester project for third semester. In this I tried to make a small scale clone of git, with functionalities such as staging changes, commiting changes, logging, and rolling back. I tried to create my own algorithm that performs delta comparisions between "previous" state of a file and the "current" state of the file to find lines deleted and lines added.
